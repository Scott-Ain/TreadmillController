## Simple prototype WinForms Treadmill FTMS controller.

Download the code and compile in something like Visual Studio 2026 with .Net core 10 or browse to the Release folder and download the exe and 2 dll's and run on a windows machine with the .Net 10 desktop runtime installed. (Download from Microsoft.)

Designed and tested to run with the https://github.com/ball2jh/Hyperborea NordicTrack treadmill controller running on the factory Android console.
